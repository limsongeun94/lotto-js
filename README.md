## 프로젝트 소개
직접 포함하고 싶은 숫자와 제외하고 싶은 숫자를 선택해 로또 번호를 추천받을 수 있습니다.

## 사용 방법
포함할 번호와 제외할 번호를 체크박스로 선택한 후 한 게임을 원하면 '한 줄 추출' 버튼을 다섯 게임을 원하면 '다섯 줄 추출' 버튼을 클릭합니다.
